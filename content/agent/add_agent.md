### What are Agents?
In Unit Master, Agents are people who introduce clients to invest in your organisationâ€™s funds. They help bring in new members and support client onboarding and account management.

The Agents List shows all agents registered in the system. From this section, you can add new agents, update their details, or change the agent assigned to a client.

### 1. **Agents List**

![text](/images/agent_list.png)

```
The Agents List displays all agents currently registered in Unit Master. It allows you to:
* View existing agents and their details
* Add new agents to the system
* Edit or update agent information

```




### 2. **Adding New Agent**

To add a new agent:
```
  Go to the Agents section from the sidebar.
 ```
<img src="/images/agents_sidebar.png" alt="Agents sidebar" class="img-small"  width= "400px"/>


### 3. **Changing a client's agent**
```
* Select the client from the client list.
* Navigate to their account details.
* Change the assigned agent from the dropdown list.
* Save the changes.
```
