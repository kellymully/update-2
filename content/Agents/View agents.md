### View Agents

***Step One***
```
Go to the Agents menu then click on the view agents search sub-menu. It gives a list of the funds’ agents.
```

 ***Step Two***
```
A user can edit the agent’s details by clicking the blue edit button.
```

***Step Three***
```
A user can also view the agent’s details by clicking on the orange view details button.

```

***Step Four***

```
A user can also delete an agent by clicking in the red delete button.
```
