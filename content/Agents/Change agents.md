### Change of Agent

***Step One***
```
Go to the Agents menu then click on the change agent sub-menu. Search the agent you want. A list of members under the agent will appear at the bottom then click the blue change agent button. 
```

***Step two***
```
The user selects the agent that a client is being transferred from and also selects the agent the client is being transferred to.
```
***Step three***
```
Once the agents are selected a list of clients under the existing agents is displayed and the user can the click on the change button to change the agent.


```
