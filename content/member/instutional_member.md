  ### Institutional/Group member registration guide

**How to register an institutional or group member**

Welcome to the **New Member Registration Guide**This page explains how to register an **Institutional or Group Member** in the system. Follow each step carefully to complete the registration successfully.

  ### 1. How to Add a New Institutional or Group Member

  For institutional and group members, the process is similar to a single member registration but with additional information about the organisation or group.



  ***step 1: Group details***
  
  This section collects basic information about the organisation or group.
    ![](/images/inst.png)

  
  **Group Member Name(Required)**
  
   Enter the full name of the group or institution.  For example: Miriam Keli Muli.
   This field cannot be left blank.
  
   **1.Industry (Required)**
   
 ```
    Select the industry the group or organisation belongs to from the dropdown list. For example: ICT, Education, Health.
 ```
   
   **2.Date of Incorporation / Registration (Required)**

   ```
  Enter the official date when the group was registered or incorporated. Use the date picker to select the date in DD-MM-YYYY format. For example: 12-05-2015.
```
**3.Institution Type (Required)**
```
Select the type of institution. Typical options include: School, Company, NGO, and others.
```
**4.Category (Required)**
```
Select whether the member is a Group or a Company by choosing the appropriate radio button.
```
**5.Registration Number (Required)**
```
Enter the official registration number issued by the relevant authority. For example: 12345678.
```

***Step 2: Contact details***

Enter the main contact and communication details for the institution or group.

![](/images/cont.png)


**1. Postal Address (Optional)**
```
*Enter the full postal address where correspondence can be sent.
```
**2.Postal Code (Optional)**
```
*Select or enter the postal code to identify the mailing area.
```
**3.Country of Residence (Required)**
```
*Choose the country where the institution or group is based.
```
**4.District (Required)**
```
*Select the district from the list provided for regional categorisation.
```
**5.Town (Required)**
```
*Select the town of residence. In some setups the default may be “Dar es Salaam”.
```
**6.Mobile Phone Number (Required)**
```
*Enter the main mobile phone number**, including country code. Used for communication and SMS notifications.
```
**7.Telephone Number (Optional)**
```
*Enter a landline or additional phone number if available.
```
**8.Email Address (Required)**
```
*Enter a valid email address. This is used for notifications and statements.
```
**9.Accept SMS Notifications? (Required)**
```
*Choose Yes if the institution should receive SMS notifications, or No otherwise.
```
**10.Send SMS Queries? (Required)**
```
*Choose whether the institution can receive SMS queries from the system.
```
**11.Receive Statement Via? (Required)**

 Choose how statements are delivered:
  ```
  * Email – statements are sent to the registered email (recommended).
  * Postage Mail – printed statements are posted and may incur a fee.
 ```
**12.Website URL (Optional)**
```
*Enter the organisation or group website URL, if applicable.
```
**13.Physical Address (Required)**
 ```
*Enter the full physical location of the institution or business. This is required for official documentation and correspondence.
```
 **14.Account Mandate (Required)**
 ```
 *Select the account mandate type that defines who is authorised to sign or approve transactions on behalf of the institution.
```
 ***Step 3: Statutory details***

 Provide statutory and compliance information for the institutional or group member.

 ![](/images/stat.png)

 **1. Client Residency and Citizenship**

```
*Is client resident? (Required) Select Yes if the client resides in the country, or No otherwise.
*Has dual citizenship? (Required) Select Yes if the client holds more than one citizenship, or No.
*Is client a US citizen? (Required) Select Yes or No as required for tax compliance.

```
**2. Tax Information**
```
*Tax No (Required) Enter the tax identification number for the institution.
*Is client tax exempt? (Required) Select Yes if the institution qualifies for tax exemption, otherwise select No.
*Exempt Certificate Expiry Date If tax exempt, enter the expiry date of the exemption certificate in DD-MM-YYYY format.
```
**3. Compliance and Screening**
```
*Has Adverse Media Screening been done? (Required) Select Yes if adverse media screening has been completed, or No otherwise.
*Has Group / Sub Members (Required) Select Yes if the client is part of a group or has sub-members.
*Group Member Logo Upload the logo for the group or institution (optional).
*Is member a PEP? Indicate whether the client is a Politically Exposed Person.
*What is the screening status? (Required) Select True if screening is complete and satisfactory, or False otherwise.
```
**4. Account Preferences**
```
*Do you want to withdraw Monthly Interest? Select Yes if interest should be withdrawn monthly, or No to reinvest it.
*Is this a suspense account? Select Yes if the account is temporary or pending verification, or No otherwise.
```
**5. Document Handling**
```
*Client sign-off date (Required) Enter the date when the client signed the physical documents (DD-MM-YYYY).
*Admin received date (Required) Enter the date when the admin received the documents (DD-MM-YYYY).
*Risk Assessment completed date (Required) Enter the date when the risk assessment was finalised (DD-MM-YYYY).
```

***Step 4. Account details***

Capture the account setup information for the institutional or group member.

![](/images/acco.png)

**1. Fund Name (Required)**
```
*Select the investment fund from the list. This defines where the member's money will be invested.
```
**2. Agent Name (Required)**
```
*Select the agent responsible for managing this account.
```
**3 .Fund Description**
```
*Describe the fund's goal, strategy, or focus. For example: "A growth fund focused on technology stocks."
```

**4. Show Tax on Statement? (Required)**
```
*Choose Yes to show a detailed tax breakdown on statements, or No to show net figures only.
```
**5. Note / Instructions**
```
*Enter any internal notes or special instructions for staff, such as: "Contact client before processing large withdrawals."
```
**6. Fee Category (Required)**
```
*Select the applicable fee category, for example Retail, Institutional, or Premium.
```
**Withdraw Dividends? (Required)**
```
*Select Yes to pay dividends out in cash, or No to reinvest dividends back into the fund.
```
