> ### Step One
```
Go to Operations menu then click on the cancel transaction sub-menu. This allows a user to cancel a transaction that has been posted in the system. The user is able to select the transaction using the transaction date. The user can upload a file to cancel transaction. 
```
![](/images/cancel_tran.PNG)

> ### Step Two
```
The user can use the transaction number to search for the transaction to be cancelled. A user then clicks on the yellow button to initiate the transaction. One can cancel multiple transactions or a single transaction.
``` 
![](/images/cancel2.PNG)

> ### Step Three
```
When cancelling multiple transactions, a tab will appear where one keys in the reason for cancellation. 
```
![](/images/cancel3.PNG)
