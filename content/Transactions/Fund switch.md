 ##  EOD Processing and fund switch

 #### Fund switch
 
 The **Fund Switch** function allows the user to move funds from one fund to another within the same member account.

 ***Step 1: Open the fund switch page***

 Go to the Transactions menu, then click on the Fund Switch sub menu.
![](/images/trans.png)


 ***Step 2: Select the member***

 Initiate a switch by:
```
 
 *Searching for the member using the Search button on the top left side.
* Selecting the memberâ€™s details from the dropdown button on the top right side.

```
![](/images/swich.png)
***Step 3: Choose date and accounts***

Select:
```
*The transaction date.
*The account funds are moving from.
*The account funds are moving to.

```
![](/images/detail.png)

The user can also create a new account for the member by clicking the New Account button. A pop up appears and the user populates the fields with the required details.

***Step 4: select switch type and reason***

Select the switch type:
```
   *Move all funds, or
   *Move partial funds.
   ```
   ![](/images/narrate.png)

   Enter a reason for the switch and, if needed, add a short narrative explaining the reason.

   ***Step 5:Submit and confirm***

   Click the **Submit** button. A confirmation window appears. Once the user confirms the switch, the transaction is complete.
