### EOD Processing and Fund Switch

**EOD Processing**

**EOD** stands for **End of Day.** EOD processing is designed to tie up all the operations for a financial day and prepare the system for the next day.

A user can run EOD for a single client or for all clients. It also allows the user to run the EOD process for a single date or for a selected period.

***Step:1 Open the EOD  processing page***
**

Go to the **Operations** menu, then click on the **EOD Processing** sub menu.

<!-- ![](/images/opere.png) -->
<img src="/images/opere.png" alt="Agents sidebar" class="img-small"  width= "400px"/>

***Step:2 Search the member***

Use the Search button at the top of the page to look for a member. Then select the member’s details from the dropdown menu below it.

![](/images/eod.png)

***Step: 3 Enter EOD parameters***

Key in the required details in the fields provided, for example:

![](/images/date.png)

   ```
   *Date – single date or period.
   *Funds to be processed.
   ```

   Once the user has set all the parameters, they click on the Compute button.

***Step: 3 Complete the EOD process***


   The user confirms the details and clicks on the Compute button to complete the EOD process.
