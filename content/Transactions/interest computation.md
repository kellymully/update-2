 > ### Interest computation
 
 This section allows the user to compute or calculate interest, validate the interest, and credit the validated interest either to an individual member or to all members.

 > ### Step One
```
Go to the operations menu then click on the Interest computation sub-menu.
```
<img src="/images/interest.png" alt="Agents sidebar" class="img-small"/>

> ### Step Two
```
Once you click on the interest Computation sub-menu the page below appears.
```
![](/images/interest2.png)

```
From the page above, the user can compute interest - for an individual client or all clients using the following parameters. Member number, all funds or single fund, by single date or period, then click on compute button.
```
> ### Step Three
```
Once the user has computed the interest, they can proceed and click on the validate interest button on the top of the page. It will give the page below.
```
![](/images/interest3.png)

```
The user can authenticate/confirm if the interest computed is correct. Interest validation can be done for all funds or a single fund. 
```
> ### Step Four

```
Once the user has validated the interest, they can proceed and click on the credit interest button (Credit Interest â€“ Individual) on the top of the page. It will give the page below.
```
![](/images/interest4.png)

```
A user can use this page to credit the computed and validated interest to a single clients account. In the case of crediting interest to all clients, the user can click on the Credit Interest (All) button which will give the page below.
```
