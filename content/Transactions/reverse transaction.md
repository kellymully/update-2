> ### Step one
```
Go to Operations menu then click on the reverse transaction sub-menu. A user is able to filter out transactions using parameters provided. Then once done click on the blue button to reverse. 
```
![](/images/reverse1.PNG)

> ### Step Two
```
A pop-up page then appears showing the details of the transaction to be reversed. The user clicks the green reverse transaction button to complete the transaction.
```
